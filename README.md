
# Simple Interest Calculator

## Project Overview
This is a simple Bash script that calculates **Simple Interest** based on user input for Principal, Rate of Interest, and Time period. 

The calculator uses the formula:

**SI = (Principal × Rate × Time) / 100**

## How to Use
1. Open a terminal.
2. Navigate to the folder containing `simple-interest.sh`.
3. Run the script:
   ```bash
   ./simple-interest.sh
